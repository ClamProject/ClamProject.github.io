Open Source

The CLAM project is an open source software project.
Anyone can review the code and contribute to the project.

Peer-to-Peer

The CLAM project is a peer-to-peer network.
Our users own and operate the code and software.

Decentralized

The CLAM project is decentralized.
Power rests in the hands of our users.

Democratized

The CLAM project is democratized.
Users can influence the future of CLAMs.
